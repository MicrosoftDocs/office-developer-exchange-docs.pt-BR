## <a name="designing-your-application-with-schema-version-in-mind"></a><span data-ttu-id="b0578-120">Criando seu aplicativo com a versão do esquema em mente</span><span class="sxs-lookup"><span data-stu-id="b0578-120">Designing your application with schema version in mind</span></span>

<span data-ttu-id="b0578-121">Tenha em mente os seguintes pontos ao projetar seu aplicativo para trabalhar com versões diferentes do esquema do EWS:</span><span class="sxs-lookup"><span data-stu-id="b0578-121">Keep the following points in mind as you design your application to work with different versions of the EWS schema:</span></span>
  
- <span data-ttu-id="b0578-122">Ativar/desativar a funcionalidade com base na versão do esquema.</span><span class="sxs-lookup"><span data-stu-id="b0578-122">Turn on/off functionality based on the schema version.</span></span> <span data-ttu-id="b0578-123">Convém mapear a funcionalidade do cliente para a versão do esquema e, em alguns casos, para a versão do serviço.</span><span class="sxs-lookup"><span data-stu-id="b0578-123">You'll want to map client functionality to the schema version and, in some cases, to the version of the service.</span></span> <span data-ttu-id="b0578-124">O exemplo a seguir retornará um [PropertySet](https://msdn.microsoft.com/library/office/microsoft.exchange.webservices.data.propertyset%28v=exchg.80%29.aspx) com base na versão do esquema e no serviço.</span><span class="sxs-lookup"><span data-stu-id="b0578-124">The following example will return a [PropertySet](https://msdn.microsoft.com/library/office/microsoft.exchange.webservices.data.propertyset%28v=exchg.80%29.aspx) based on the version of the schema and service.</span></span> 
    
  ```cs
  private static PropertySet InitPropertySetByVersion(ExchangeService service)
  {
      PropertySet props;
      // The schema version to target to access the NormalizedBody property 
      // is Exchange2013 or later. The server version to target to access the 
      // NormalizedBody property on an email is 15 or later, which 
      // equates to Exchange 2013.
      if (service.RequestedServerVersion >= ExchangeVersion.Exchange2013 &amp;&amp;
          service.ServerInfo.MajorVersion >= 15)
      {
          props = new PropertySet(EmailMessageSchema.NormalizedBody);
      }
      else
      {
          props = new PropertySet(EmailMessageSchema.Body);
      }
      return props;
  }
  ```
