# <a name="onlinemeetingsettings"></a>OnlineMeetingSettings

O elemento **OnlineMeetingSettings** Especifica as configurações para reuniões online. 
  
```XML
<OnlineMeetingSettings>
   <LobbyBypass/>
   <AccessLevel/>
   <Presenters/>
</OnlineMeetingSettings>
```

 **OnlineMeetingSettingsType**
## <a name="attributes-and-elements"></a>Attributes and elements

As seções a seguir descrevem os atributos e elementos filho elementos pai.
  
### <a name="attributes"></a>Atributos

Nenhum.
  
### <a name="child-elements"></a>Elementos filho

[LobbyBypass](lobbybypass.md) | [AccessLevel](accesslevel.md) | [apresentadores](presenters.md)
  
### <a name="parent-elements"></a>Elementos pai

[CalendarItem](calendaritem.md)
