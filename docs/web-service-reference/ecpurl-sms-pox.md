# <a name="ecpurl-sms-pox"></a>EcpUrl-sms (POX)

O elemento **EcpUrl-sms** Especifica uma URL parcial que pode ser combinada com o valor do elemento [EcpUrl POX ()](ecpurl-pox.md) para gerar uma URL que pode ser usada para acessar as configurações do serviço SMS (Short Message) para um usuário habilitado para email.

```XML
<EcpUrl-sms/>
```

## <a name="text-value"></a>Text value

O valor de texto representa uma URL parcial que pode ser combinada com o valor do elemento [EcpUrl POX ()](ecpurl-pox.md) para gerar uma URL que pode ser usada para acessar as configurações do SMS para o usuário.
