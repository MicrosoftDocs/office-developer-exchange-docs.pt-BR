# <a name="alternatemailboxes-soap"></a>AlternateMailboxes (SOAP)

O elemento **AlternateMailboxes** representa uma coleção de caixas de correio alternadas. 
  
```XML
<AlternateMailboxes>
   <AlternateMailbox/>
</AlternateMailboxes>
```

 **AlternateMailboxes**
## <a name="attributes-and-elements"></a>Atributos e elementos

As seções a seguir descrevem os atributos, os elementos filhos e os elementos pai.
  
### <a name="attributes"></a>Atributos

Nenhum
  
### <a name="child-elements"></a>Elementos filho

|**Elemento**|**Descrição**|
|:-----|:-----|
|[AlternateMailbox (SOAP)](alternatemailbox-soap.md) <br/> |Representa uma caixa de correio alternativa.  <br/> |
   
### <a name="parent-elements"></a>Elementos pai

|:-----|:-----|
|[AlternateMailboxCollectionSetting (SOAP)](alternatemailboxcollectionsetting-soap.md) <br/> |Contém uma coleção de configurações de caixa de correio alternativas.  <br/> |
