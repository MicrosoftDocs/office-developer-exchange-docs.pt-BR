# <a name="alternatemailboxes-soap"></a>AlternateMailboxes (SOAP)

O elemento **AlternateMailboxes** representa uma coleção de caixas de correio alternativas. 
  
```XML
<AlternateMailboxes>
   <AlternateMailbox/>
</AlternateMailboxes>
```

 **AlternateMailboxes**
