# <a name="responsecode"></a>ResponseCode

O elemento **ResponseCode** fornece informações de status sobre a solicitação. 
  
- [ResponseMessage](responsemessage.md) 
- [ResponseCode](responsecode.md)
  
```XML
<ResponseCode/>
```

**ResponseCodeType**
