# <a name="protocolconnections-soap"></a>ProtocolConnections (SOAP)

O elemento **ProtocolConnections** contém as conexões de protocolo de zero ou mais. 
  
```XML
<ProtocolConnections>
   <ProtocolConnection/>
</ProtocolConnections>
```
