# <a name="term"></a>Termo

O elemento de **termo** Especifica um termo realçado em uma resposta **FindConversation** ou **FindItem** . 
  
```XML
<Term>
   <Scope/>
   <Value/>
</Term>
```

 **HighlightTermType**
## <a name="attributes-and-elements"></a>Attributes and elements

As seções a seguir descrevem os atributos e elementos filho elementos pai.
