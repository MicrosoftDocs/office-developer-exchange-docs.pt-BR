# <a name="getfederationinformationrequest-soap"></a>GetFederationInformationRequest (SOAP)

O elemento **GetFederationInformationRequest** representa os parâmetros de uma chamada para a operação [GetFederationInformation operação (SOAP)](getfederationinformation-operation-soap.md).
  
```XML
<GetFederationInformationRequest>
   <Domain/>
</GetFederationInformationRequest>
```

**GetFederationInformationRequest**
