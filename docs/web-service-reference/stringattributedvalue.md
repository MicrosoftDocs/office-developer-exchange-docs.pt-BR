# <a name="stringattributedvalue"></a>StringAttributedValue

O elemento **StringAttributedValue** Especifica uma instância em uma matriz de atributos associados a um elemento de pessoa. 
  
```XML
<StringAttributedValue>
   <Value/>
   <Attributions/>
</StringAttributedValue>
```

 **StringAttributedValueType**
