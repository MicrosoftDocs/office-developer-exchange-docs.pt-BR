# <a name="assigncategories"></a>AssignCategories

O elemento **AssignCategories** representa as categorias que estão marcadas em mensagens de email. 
  
- [Regra (RuleType)](rule-ruletype.md)
  
- [Ações](actions.md)
  
```XML
<AssignCategories>
   <String/>
</AssignCategories>
```

 **ArrayOfStringsType**
## <a name="attributes-and-elements"></a>Attributes and elements

As seções a seguir descrevem os atributos e elementos filho elementos pai.
  
### <a name="attributes"></a>Atributos

Nenhum.
  
### <a name="child-elements"></a>Elementos filho

|**Elemento**|**Descrição**|
|:-----|:-----|
|[String](string.md) <br/> |Contém uma cadeia de caracteres que identifica uma única categoria.  <br/> |
   
### <a name="parent-elements"></a>Elementos pai

|**Elemento**|**Descrição**|
|:-----|:-----|
|[Ações](actions.md) <br/> |Representa o conjunto de ações que estão disponíveis a ser executada em uma mensagem quando as condições são atendidas.  <br/> |
   
## <a name="text-value"></a>Text value

Nenhum.
  
## <a name="remarks"></a>Comentários

O esquema que descreve este elemento está localizado no diretório virtual do IIS que hospeda os Serviços Web do Exchange.
