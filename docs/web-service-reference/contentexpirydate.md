# <a name="contentexpirydate"></a>ContentExpiryDate

O elemento **ContentExpiryDate** Especifica a data de validade do conteúdo. 
  
```XML
<ContentExpiryDate></ContentExpiryDate>
```

 **cadeia de caracteres**
## <a name="attributes-and-elements"></a>Attributes and elements

As seções a seguir descrevem os atributos e elementos filho elementos pai.
  
### <a name="attributes"></a>Atributos

Nenhum.
  
### <a name="child-elements"></a>Elementos filho

Nenhum.
  
### <a name="parent-elements"></a>Elementos pai

|**Elemento**|**Descrição**|
|:-----|:-----|
|[RightsManagementLicenseData](rightsmanagementlicensedata.md) <br/> |Especifica informações sobre a licença de gerenciamento de direitos.  <br/> |
   
## <a name="text-value"></a>Text value

O valor de texto do elemento **ContentExpiryDate** é uma cadeia de caracteres que especifica a data de validade do conteúdo. 
  
## <a name="remarks"></a>Comentários

Este elemento foi introduzido no Exchange Server 2013.
  
O esquema que descreve este elemento está localizado no diretório virtual do IIS que hospeda os Serviços Web do Exchange.
