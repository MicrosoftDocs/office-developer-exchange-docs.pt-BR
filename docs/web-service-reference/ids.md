# <a name="ids"></a>IDs

O elemento de **Ids** contém uma matriz de identificadores de definição de fuso horário. 
  
```XML
<Ids>
   <Id/>
</Ids>
```

 **NonEmptyArrayOfTimeZoneIdType**
## <a name="attributes-and-elements"></a>Attributes and elements

As seções a seguir descrevem os atributos e elementos filho elementos pai.
  
### <a name="attributes"></a>Atributos

Nenhum.

|**Elemento**|**Descrição**|
|:-----|:-----|
|[ID (TimeZone)](id-timezone.md) <br/> |O elemento que identifica a definição de um único fuso horário.  <br/> |
   
### <a name="parent-elements"></a>Elementos pai

|:-----|:-----|
|[GetServerTimeZones](getservertimezones.md) <br/> |Define uma solicitação para recuperar as definições de fuso horário do servidor Exchange.  <br/> |
