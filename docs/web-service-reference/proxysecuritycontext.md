# <a name="proxysecuritycontext"></a>ProxySecurityContext

O elemento **ProxySecurityContext** é usado pelo proxy HTTP do computador que está executando o Microsoft Exchange Server 2007 que tenha acesso para cliente a função de servidor instalada e não é usada por operações de serviços Web do Exchange. Este elemento foi introduzido no Exchange Server 2007 Service Pack 1 (SP1). 
  
```xml
<ProxySecurityContext/>
```

 **ProxySecurityContextType**
## <a name="attributes-and-elements"></a>Attributes and elements

As seções a seguir descrevem os atributos e elementos filho elementos pai.
  
### <a name="attributes"></a>Atributos

Nenhum.
  
### <a name="child-elements"></a>Elementos filho

Nenhum.
  
### <a name="parent-elements"></a>Elementos pai

Nenhum.
  
## <a name="remarks"></a>Comentários

O esquema que descreve este elemento está localizado no diretório virtual EWS do computador que está executando o Microsoft Exchange Server 2007 que possui a função de servidor acesso para cliente instalada.
