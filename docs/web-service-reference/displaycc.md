# <a name="displaycc"></a>DisplayCc

O elemento **DisplayCc** representa a cadeia de caracteres de exibição que é usada para o conteúdo da caixa Cc. Esta é a cadeia de caracteres concatenada de todos os nomes de exibição destinatários Cc. 
  
```xml
<DisplayCc/>
```

 **String**
## <a name="attributes-and-elements"></a>Attributes and elements

As seções a seguir descrevem os atributos e elementos filho elementos pai.
  
### <a name="attributes"></a>Atributos

Nenhum.
  
### <a name="child-elements"></a>Elementos filho

Nenhum.

## <a name="text-value"></a>Text value

O valor de texto representa a cadeia de caracteres de exibição que é usada para o conteúdo da caixa Cc.
