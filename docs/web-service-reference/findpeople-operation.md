## <a name="findpeople-operation-request-example"></a><span data-ttu-id="f8804-130">Exemplo de solicitação de operação FindPeople</span><span class="sxs-lookup"><span data-stu-id="f8804-130">FindPeople operation request example</span></span>

<span data-ttu-id="f8804-131">O exemplo a seguir de uma solicitação de operação **FindPeople** mostra como retornar os primeiros 100 contatos da pasta contatos.</span><span class="sxs-lookup"><span data-stu-id="f8804-131">The following example of a **FindPeople** operation request shows how to return the first 100 contacts from the Contacts folder.</span></span> 
  
```XML
<?xml version="1.0" encoding="UTF-8"?>
<soap:Envelope xmlns:soap="http://schemas.xmlsoap.org/soap/envelope/"
               xmlns:t="https://schemas.microsoft.com/exchange/services/2006/types"
               xmlns:m="https://schemas.microsoft.com/exchange/services/2006/messages">
   <soap:Header>
      <t:RequestServerVersion Version="Exchange2013" />
   </soap:Header>
   <soap:Body >
      <m:FindPeople>
         <m:IndexedPageItemView BasePoint="Beginning" MaxEntriesReturned="100" Offset="0"/>
         <m:ParentFolderId>
            <t:DistinguishedFolderId Id="contacts"/>
         </m:ParentFolderId>
      </m:FindPeople>
   </soap:Body>
</soap:Envelope>
```

<span data-ttu-id="f8804-132">O corpo SOAP de solicitação contém os seguintes elementos:</span><span class="sxs-lookup"><span data-stu-id="f8804-132">The request SOAP body contains the following elements:</span></span>
  
- [<span data-ttu-id="f8804-133">FindPeople</span><span class="sxs-lookup"><span data-stu-id="f8804-133">FindPeople</span></span>](findpeople.md)
    
- [<span data-ttu-id="f8804-134">IndexedPageItemView</span><span class="sxs-lookup"><span data-stu-id="f8804-134">IndexedPageItemView</span></span>](indexedpageitemview.md)
    
- [<span data-ttu-id="f8804-135">ParentFolderId (TargetFolderIdType)</span><span class="sxs-lookup"><span data-stu-id="f8804-135">ParentFolderId (TargetFolderIdType)</span></span>](parentfolderid-targetfolderidtype.md)
    
- [<span data-ttu-id="f8804-136">DistinguishedFolderId</span><span class="sxs-lookup"><span data-stu-id="f8804-136">DistinguishedFolderId</span></span>](distinguishedfolderid.md)
    
<span data-ttu-id="f8804-137">O exemplo a seguir de uma solicitação de operação **FindPeople** mostra como retornar os primeiros 100 contatos da GAL usando uma cadeia de caracteres de consulta.</span><span class="sxs-lookup"><span data-stu-id="f8804-137">The following example of a **FindPeople** operation request shows how to return the first 100 contacts from the GAL by using a query string.</span></span> <span data-ttu-id="f8804-138">A configuração do **DistinguishedFolderId** como "Directory" pesquisará a GAL como a principal fonte de personas.</span><span class="sxs-lookup"><span data-stu-id="f8804-138">Setting the **DistinguishedFolderId** to "directory" will search the GAL as the primary source of personas.</span></span> 
  
```XML
<?xml version="1.0" encoding="UTF-8"?>
<soap:Envelope xmlns:soap="http://schemas.xmlsoap.org/soap/envelope/"
               xmlns:t="https://schemas.microsoft.com/exchange/services/2006/types"
               xmlns:m="https://schemas.microsoft.com/exchange/services/2006/messages">
   <soap:Header>
    <t:RequestServerVersion Version="Exchange2013" />
   </soap:Header>
   <soap:Body >
    <m:FindPeople>
      <m:PersonaShape>
        <t:BaseShape>IdOnly</t:BaseShape>
        <t:AdditionalProperties>
          <t:FieldURI FieldURI="persona:DisplayName"/>
          <t:FieldURI FieldURI="persona:Title"/>
        </t:AdditionalProperties>
      </m:PersonaShape>
      <m:IndexedPageItemView BasePoint="Beginning" MaxEntriesReturned="100" Offset="0"/>
      <m:ParentFolderId>
        <t:DistinguishedFolderId Id="directory"/>
      </m:ParentFolderId>
      <m:QueryString>adams</m:QueryString>
    </m:FindPeople>
  </soap:Body>
</soap:Envelope>
```
