# <a name="isdeliveryreceiptrequested"></a>IsDeliveryReceiptRequested

O elemento **IsDeliveryReceiptRequested** indica se o remetente de um item solicita uma confirmação de entrega. 
  
```xml
<IsDeliveryReceiptRequested/>
```

 **Boolean**
## <a name="attributes-and-elements"></a>Attributes and elements

As seções a seguir descrevem os atributos e elementos filho elementos pai.
  
### <a name="attributes"></a>Atributos

Nenhum.
  
### <a name="child-elements"></a>Elementos filho

Um valor de texto de **true** indica que uma confirmação de entrega é solicitada do destinatário do item.
