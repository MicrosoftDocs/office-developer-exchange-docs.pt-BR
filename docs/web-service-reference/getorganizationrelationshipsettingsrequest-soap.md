# <a name="getorganizationrelationshipsettingsrequest-soap"></a>GetOrganizationRelationshipSettingsRequest (SOAP)

O elemento **GetOrganizationRelationshipSettingsRequest** representa os parâmetros de uma chamada para a operação de [operação do GetOrganizationRelationshipSettings (SOAP)](getorganizationrelationshipsettings-operation-soap.md) . O elemento **GetOrganizationRelationshipSettingsRequest** é somente para uso interno. Esse elemento não é usado por clientes. 
  
```XML
<GetOrganizationRelationshipSettingsRequest>
   <Domains/>
</GetOrganizationRelationshipSettingsRequest>
```

 **GetOrganizationRelationshipSettingsRequest**
