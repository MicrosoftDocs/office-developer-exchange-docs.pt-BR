# <a name="getorganizationrelationshipsettingsrequest-soap"></a>GetOrganizationRelationshipSettingsRequest (SOAP)

O elemento **GetOrganizationRelationshipSettingsRequest** representa os parâmetros de uma chamada para a operação [GetOrganizationRelationshipSettings operação (SOAP)](getorganizationrelationshipsettings-operation-soap.md) . O elemento **GetOrganizationRelationshipSettingsRequest** é apenas para uso interno. Este elemento não é usado pelos clientes. 
  
```XML
<GetOrganizationRelationshipSettingsRequest>
   <Domains/>
</GetOrganizationRelationshipSettingsRequest>
```
