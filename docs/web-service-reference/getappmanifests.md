# <a name="getappmanifests"></a>GetAppManifests

O elemento **GetAppManifests** é a base para uma solicitação retornar o manifesto para aplicativos. 
  
```xml
<GetAppManifests />
```

 **GetAppManifestsType**
## <a name="attributes-and-elements"></a>Attributes and elements

As seções a seguir descrevem os atributos e elementos filho elementos pai.
  
### <a name="attributes"></a>Atributos

Nenhum.
  
### <a name="child-elements"></a>Elementos filho

[ApiVersionSupported](apiversionsupported.md) | [SchemaVersionSupported](schemaversionsupported.md)
  
### <a name="parent-elements"></a>Elementos pai

Nenhum.
  
## <a name="text-value"></a>Text value

Nenhum.
  
## <a name="remarks"></a>Comentários

O elemento **GetAppManifests** é aplicável para clientes que visam o Exchange Online e versões do Microsoft Exchange Server, começando com o Exchange 2013.
